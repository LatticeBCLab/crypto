linkable ring signature
=============================================
Linkable Ring Signature（可链接环签名）是一种数字签名方案，它允许签名者在一个由多个公钥组成的集合中匿名签名，并且可以保证签名者的匿名性和签名的可链接性。
这意味着可以通过检查两个签名是否使用了同一个密钥来确定两个签名的签名者是否为同一个人，同时其他人无法知道签名者的身份。
